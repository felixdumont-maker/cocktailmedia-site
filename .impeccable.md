## Design Context

### Users
PME de la Mauricie (Trois-Rivières, Shawinigan, région) — propriétaires d'entreprise, responsables marketing, indépendants. Ils cherchent du contenu pro (photo, vidéo, graphisme, web) sans passer par une agence lourde. Contexte : bureau ou téléphone, décision rapide, budget PME.

### Brand Personality
**Franc, accessible, artisan.**
- Transparent : prix affichés, aucune surprise, aucun intermédiaire
- Direct : tu choisis, tu paies, on livre — pas de comité, pas de réunion
- Local : basé à Trois-Rivières, ancré en Mauricie

### Aesthetic Direction
- **Thème** : Light — produit consulté en journée, au bureau, par des gens qui prennent des décisions pratiques
- **Palette** : Fond chaud (--fond #FAF9F6 / --creme #FFF4E9), noir profond (--noir #2B2B2B), rouge vif (--rouge #E83B14) comme seul accent
- **Typographie** : Bebas Neue (display, headings, labels) + Montserrat (body) — duo percutant et lisible
- **Ton visuel** : Menu de restaurant — chaque section est une carte, les prix sont affichés, le parcours est évident
- **Anti-références** : pas d'agence froide, pas de glassmorphism, pas de gradient text, pas de dark mode techno

### Design Principles
1. **Prix visibles partout** — le prix est le message, jamais caché
2. **Pas de bruit superflu** — chaque élément gagne sa place, spacing généreux
3. **Rouge avec parcimonie** — accent fort sur les CTA, titres choc, prix, points actifs uniquement
4. **Hiérarchie Bebas → Montserrat** — Bebas pour ce qui commande l'attention, Montserrat pour ce qui informe
5. **Hover = CSS, jamais JS inline** — utiliser les classes `.btn-*`, `.card`, `.faq-item`, `.social-btn`, `.footer-link`

### Design System — composants clés
- `components/ui/Btn.tsx` — boutons/liens pill avec variants : primary, secondary, ghost-dark, white-on-rouge, outline-white, creme-on-dark
- `components/ui/SectionHeader.tsx` — eyebrow + heading + subtitle optionnel
- `app/globals.css` — tokens CSS, classes utilitaires (.container, .btn-*, .card, .faq-item, .field-*, .footer-link, .social-btn, .eyebrow, .heading-display-*)
